# granola-kit

MCP server and CLI for Granola.ai's private API: meeting notes, transcripts, chat-based meeting Q&A, folders, people &
companies, workspaces, and account.

**Status: under construction** — built as vertical slices, each proven against the live API via the CLI. Reconnaissance
and the implementation plan live in [`claude-docs/granola-kit/`](../../claude-docs/granola-kit/).
